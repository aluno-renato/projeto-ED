# 📦 Sistema de Controle de Pedidos para Delivery

## 📌 Descrição

Este projeto consiste no desenvolvimento de um sistema em Java para
gerenciamento de pedidos de uma empresa de delivery, utilizando
**estruturas de dados encadeadas implementadas manualmente**.

O sistema permite o controle completo do fluxo de pedidos, desde o
cadastro de clientes até o histórico de entregas, garantindo
organização, eficiência e rastreabilidade das operações.

------------------------------------------------------------------------

## 🎯 Objetivos

-   Gerenciar clientes e produtos
-   Criar e controlar pedidos
-   Organizar pedidos em fila de preparo
-   Registrar histórico de pedidos concluídos
-   Permitir navegação entre pedidos ativos
-   Aplicar conceitos de estruturas de dados na prática

------------------------------------------------------------------------

## 🧠 Estruturas de Dados Utilizadas

### 🔹 Lista Simplesmente Encadeada

-   Lista de clientes
-   Lista de produtos
-   Itens dentro de um pedido

### 🔹 Lista Duplamente Encadeada

-   Pedidos ativos
-   Navegação entre pedidos

### 🔹 Fila Encadeada (FIFO)

-   Fila de preparação dos pedidos

### 🔹 Pilha Encadeada (LIFO)

-   Histórico de pedidos concluídos
